#eyerest

eyerest是一款通过定时锁定屏幕，提醒用户休息的小软件

##功能

* 定时锁定屏幕一定时间
* 根据是否有鼠标与键盘事件检测用户是否离开
* 可配置间隔时间，锁屏时间，空闲时间
* 可配置锁屏时屏幕上显示的文字，位置，字体，大小与颜色
* 提供dbus接口，可以进行暂停，推迟等

##编译与运行

###依赖

* x11
* xtst
* xft
* glib-2.0
* gio-2.0

###编译

进入代码目录，然后执行：

    cmake .
    make

如果想安装，也可以执行：

    make install

###运行

直接执行`eyerest-daemon`

源代码目录中有一份配置文件示例，可以拷贝到`$HOME/.config/eyerest.conf`中进行修改，修改完成后需要重启软件。
